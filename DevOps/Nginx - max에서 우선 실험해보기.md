# 설치
> brew install nginx

# 설정
nginx를 설치하면 기본적으로 포트가 8080으로 잡혀있다. 포트번호를 80으로 설정해보자

- nginx.cof 파일 수정하기
  - sudo vi/opt/homebrew/etc/nginx/nginx.conf
- 8080 -> 80으로 수정
